# Özgür Kafe Menü

Özgür Kafe Edirne'nin dijital menü sayfası. QR kod ile erişilebilir.

## Özellikler

- Modern ve responsive tasarım
- Kategori filtreleme sistemi
- Splash screen animasyonu
- Placeholder görsel desteği

## Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla)

## Kullanım

Live Server veya benzeri bir local server ile çalıştırılabilir.
